# Demo

A SwiftUI demo app showcasing multiple core frameworks, with a current focus on the CorePresentation Banner system.

## Requirements

- iOS 26.0+
- Xcode 26.0+
- Swift 6.0+

## Features

- **Demo Tab**: Interactive showcase of Banner functionality
- **Testing Tab**: Automated tests with export capability

## Setup

The Demo app depends on `CoreSwift` and `CorePresentation` frameworks. These must be downloaded and placed in the `Frameworks/` directory before opening the project.
